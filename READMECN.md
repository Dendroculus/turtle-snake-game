# 贪吃蛇游戏

经典贪吃蛇游戏，使用 Python 的 `turtle` 模块开发。游戏特色包括多种水果类型、音效以及动态记分板。

## 游戏玩法

- 使用 **W** 上移，**S** 下移，**A** 左移，**D** 右移  
- 吃水果让蛇变长，同时增加分数  
- 避免碰到墙壁或自己的身体  

## ✨ 游戏特色
- 多种水果类型，每种分值不同  
- 实时更新的动态记分板  
- 吃水果和碰撞的音效反馈  
- 流畅的 WASD 操作体验  
- 游戏结束画面
